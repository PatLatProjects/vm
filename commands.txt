Register Calling Conventions
https://learn.microsoft.com/en-us/cpp/build/x64-calling-convention?view=msvc-170

First 4 parameters to any given function in this order
rcx rdx r8 r9
Extras will be pushed into the stack

In any given function call, rax will be the return value of the function
Same with the others on return as the callee has free rights to do whatever it wants
Other registers like rbx rsi rdi are all preserved
